# Guardrail

![Guardrail](./guardrail-app/public/guardrail.png)

A guard contract restricting `DELEGATECALL` in safe smart account.

## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Gas Snapshots

```shell
$ forge snapshot
```

### For running the app

```shell
$ npm run dev
```
